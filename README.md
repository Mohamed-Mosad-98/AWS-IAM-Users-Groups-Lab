# AWS IAM Users & Groups Lab

## 📌 Overview

This lab demonstrates the fundamentals of AWS Identity and Access Management (IAM) by creating and managing IAM users and user groups using the AWS Management Console.

---

## 🎯 Objectives

- Create IAM User Groups
- Create IAM Users
- Assign users to groups
- Verify group membership
- Understand basic IAM user management

---

## 🛠️ AWS Service Used

- AWS Identity and Access Management (IAM)

---

## 📋 Lab Tasks

### Task 1 – Create IAM User Groups

Created the following user groups:

- Developers
- DevOps
- Auditors
